# AWS Infrastructure Automation with Terraform

This project demonstrates how to deploy a modern, scalable architecture on AWS using Infrastructure as Code (IaC).

## Architecture Components
* **VPC:** Custom network with public and private subnets.
* **ALB:** Application Load Balancer to distribute traffic.
* **ASG:** Auto Scaling Group for high availability.
